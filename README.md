## Overview

**Hope Tech** is a fully responsive UI/UX design project for an interior design company targeting the Ethiopian market. It delivers a clean, modern, and culturally relevant web experience that helps users discover high-quality, affordable home furniture and decor.

Designed with love in Addis Ababa, this website combines elegant visuals, smooth interactions, and local context (Ethiopian names, Birr currency, local vibe).

## Features

| Feature                    | Description                                                                 |
|----------------------------|-----------------------------------------------------------------------------|
| Fully Responsive           | Perfectly adapts to mobile, tablet, and desktop screens                     |
| Interactive Sliders        | Slick Carousel for newest products & customer testimonials                  |
| Smart Product Filtering    | Filter by category, price, and style (sofas, lamps, tables, etc.)           |
| Smooth Animations          | Powered by GSAP for delightful micro-interactions                           |
| Ethiopian Context          | Prices in ETB, local names (Abdi Kebede, Tigist Alemayehu), Addis-based     |
| Contact Form               | Easy inquiry form for custom interior requests                              |
| Scrollspy Navigation       | Smooth scrolling navbar with active section highlight                       |
| Social Media Ready         | Facebook & Instagram links integrated                                       |
